### 📡 Radar System — IoT Embedded Project

A smart radar system built on the **ESP32-S3** microcontroller that detects nearby objects using a **Time-of-Flight (ToF) distance sensor**.  
The system integrates a **servo motor, RGB LED, buzzer, and push button**, all running concurrently using **FreeRTOS** for real-time performance.

**Tech:** C++, ESP32-S3, FreeRTOS, Embedded Systems, VS Code

### 🤖 Acoustic Leak Detection — Machine Learning Project

Developed a machine learning model during my internship at Halliburton to classify pipeline leakage types based on their acoustic signatures.

The model distinguishes between four leakage categories:
- Gas → Gas
- Gas → Liquid
- Liquid → Gas
- Liquid → Liquid

**Tech:** Python, Machine Learning, Signal Processing
